# Crisisdataset_EDA_analysis

## Global Economic Crisis — Exploratory Data Analysis (2000–2024)

A comprehensive Exploratory Data Analysis (EDA) project on the Global Economic Crisis Dataset, covering 80 countries across 24 years (2000–2024). This project uncovers patterns in GDP growth, inflation, unemployment, government debt, poverty, inequality, and banking crises across 4 income groups.

---

## 📊 Dataset Overview

| Property | Details |
|---|---|
| Rows | 26,000 |
| Countries | 80 |
| Time Period | 2000 – 2024 |
| Income Groups | High, Upper Middle, Lower Middle, Low |
| Key Features | 17 columns used for analysis |

---

## 🔍 What This Project Covers

- Data loading, inspection and cleaning
- Handling missing values using a preservation strategy (median fills for numerical, mode for categorical)
- 20+ analytical questions answered with visualizations
- Univariate, Bivariate and Multivariate analysis
- Charts used: Bar, Horizontal Bar, Line, Box Plot, Heatmap

---

## 💡 Key Insights

- **Asia-Pacific** recorded the highest banking crisis count (285 country-years)
- **Income group** is the strongest predictor of economic stability
- **Inflation spiked** in two waves — 2008 (GFC) and 2021–2023 (post-COVID)
- **Banking crises and sovereign defaults** tend to co-occur — twin crisis effect
- **High Income country debt** surged post-2008 and post-2020 due to stimulus spending
